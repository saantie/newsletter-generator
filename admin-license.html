📜 Google Apps Script — วางในชีตที่ใช้งาน
Code.gs
วิธีติดตั้ง:
1. Google Sheet → Extensions → Apps Script → ลบโค้ดเดิม → วางโค้ดด้านบน
2. แก้ ADMIN_KEY และ REG_KEY
3. Deploy → New deployment → Web app → Execute as Me → Access Anyone
4. คัดลอก URL → ใส่ช่อง "Apps Script URL" ด้านบน และใส่ใน APPS_SCRIPT_URL ในแอป (index.html)
